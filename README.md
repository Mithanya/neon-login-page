# Neon Gradient Login Page

This project is a modern and visually engaging animated login page built using HTML and CSS. It demonstrates advanced CSS techniques to create interactive UI elements without relying on JavaScript.

## Features
- Animated neon gradient border using CSS conic gradients
- Smooth hover-based expansion effect
- Clean and minimal login interface
- Responsive layout for different screen sizes
- Fully implemented using HTML and CSS only

## Technologies Used
- HTML5
- CSS3 (Flexbox, Animations, Gradients)

## Project Structure
- index.html

## How to Run
1. Clone the repository
2. Open the `index.html` file in any modern web browser
